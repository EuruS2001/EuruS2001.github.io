Clases

fundamentos a la ingenieria de software

La ingeniería de software es una aplicación practica del conocimiento científico para promover metodologías y técnicas que ayuden a desarrollar sistemas de software a tiempo y a su vez aseguren que el desarrollador cumpla con las expectativas de calidad y permanezca dentro del presupuesto

Objetivos de la Ingenieria en software

-Diseñar programas informáticos que se adecuen a las exigencias de la sociedad.

-Liderar y acopiar el desarrollo de programaciones complicadas.

-Actuar en todas las fases del ciclo de la vida de un proyecto.

-Computar los costos de un proyecto y evaluar los tiempos de desarrollo.

-Realizar el seguimiento de costos y plazos.

Proceso de ingenieria de software

Concepto de calidad: conjunto de propiedades y de características de un producto o servicio, que le confieren aptitud para satisfacer una necesidad explicita o implícita.

Concepto de software: es el grado con el que un sistema componente o proceso cumplo los requerimientos específicos y las necesidades o expectativas del cliente o usuario.

Clasificación de los factores de calidad del software

Medidas directas: Cuando disponemos de un instrumento de medida que nos muestran un resultado.

Medidas indirectas: Cuando hablamos de sistemas informáticos no siempre es posible realizar una medida directa por que nos disponemos del instrumento adecuado que nos permita realizar la medición.

Sistemas de información

Un sistema de información realiza cuatro actividades básicas:

Entrada: Es el proceso mediante el cual el sistema de información toma los datos que requieren para procesar la información.

Almacenamiento: A través de esta actividad el sistema puede recordar la información guardada en sección o proceso anterior.

Procesamiento: Es la capacidad de un sistema de información para efectuar cálculos de acuerdo con una secuencia de operaciones preestablecidos.

Salida de información: Es la capacidad de sacar la información procesada o bien datos de entrada al exterior.

Elementos de un sistema de información: Financieros, Administrativos, Humanos, Materiales, Tecnológicos.

Proceso de software

Conjunto ordenado de actividades: una serie de pasos que involucran tareas, restricciones, y recursos que producen una determinada salida esperada.

Procesos del software: Especificación, Diseño o implementación, validación, evolución.
